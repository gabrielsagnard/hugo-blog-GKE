# hugo-blog-GKE
Blog Hugo hébergé chez Google Cloud Storage

# Développer en local 

    git clone https://github.com/gabrielsagnard/hugo-blog-GKE.git
    
    cd hugo-blog-GKE
    
 Puis builder le site : 
 
    hugo 

Et enfin lancer le serveur local : 

    hugo server -D 
    
(sur http://localhost:1313)
